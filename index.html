Abstract

This minimal project shows how to build a site that looks like a LaTeX document using LaTeX.css. It demonstrates author/abstract, theorems, definitions, proofs, sidenotes, tables with custom borders and alignment, dark mode, MathJax equations, and Prism code highlighting.

Dark Mode & Typeface

Click the buttons in the header to toggle dark mode (adds/removes latex-dark on <body>) and to switch between Latin Modern and Libertinus (toggles libertinus on <body>).

Language Labels

To change theorem/definition labels to Spanish, add this line in the head and set lang="es" on <html>: 

<link rel="stylesheet" href="https://latex.vercel.app/lang/es.css" />

Built with LaTeX.css. Customize via assets/styles.css.
